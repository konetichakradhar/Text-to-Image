# Text to Image Generation

This project generates images from text prompts using deep learning models.

## Features
- Generate images from text prompts
- Uses pretrained models
- Runs on Google Colab

## How to Run
1. Open notebook.ipynb in Google Colab
2. Run all cells
3. Enter your prompt

## Tech Used
- Python
- PyTorch / Diffusers (if used)
